# QQ 指南

## 向联系人发送消息

1. 在底部栏找到联系人点击进入联系人界面
2. 在顶部搜索框内输入搜索对象
3. 找到相应对象，一般为搜索出来的第一个，点击进入聊天界面
4. 向该联系人发送相应消息

## 添加新联系人或者群聊

1. 打开QQ进入联系人页面
2. 点击右上角图标进入找人页面
3. 点击搜索框输入QQ号或者群号
4. 找到相应对象，一般为搜索出来的第一个，点击进入发送申请界面
5. 填写信息后，点击发送即为发送成功

## 浏览QQ空间的方法

1. 打开QQ进入动态页面
2. 点击空间动态，进入个人空间主页面
3. 向下滑动可浏览QQ空间
4. 点击每一篇的动态右下角三个图标，从左到右分别是“点赞”，“评论”，“转发"

## 聊天置顶和标为未读和删除聊天记录的方法

1. 打开QQ进入消息页面
2. 选择聊天框左滑出现三个选项，蓝色为”置顶“，黄色为”标为未读“，红色为”删除“
